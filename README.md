# mtools-release
# mtools — 自媒体工具箱

> 一款面向内容创作者的轻量级桌面工具集，集录屏、GIF 制作、视频快剪、AI 字幕于一体，开箱即用，无需复杂配置。

---

## 核心功能一览

### 1. 屏幕录制
支持三种录制模式，满足不同场景需求：

| 模式 | 说明 |
|---|---|
| **区域录制** | 自由框选屏幕任意区域，精准录制局部内容 |
| **窗口录制** | 自动识别并锁定应用窗口，窗口移动时录制区域跟随 |
| **全屏录制** | 一键录制整个屏幕 |

**录制特性**：
- 快捷键 `F9` 快速开始 / 停止录制
- 区域录制时显示悬浮工具栏，支持开始、暂停、恢复、结束操作
- 录制过程中实时显示红色边框提示
- 支持系统音频录制（WASAPI）
- 输出格式：MP4（H.264 / HEVC）

---

### 2. GIF 录制
将屏幕内容直接录制为 GIF 动图：
- 同样支持区域 / 窗口 / 全屏三种模式
- 可调节帧率和画质
- 适合制作教程演示、表情包、UI 交互动画

---

### 3. 视频快剪
轻量级视频剪辑工作台，专注于"快速裁剪 + 字幕处理"两大高频需求。

#### 3.1 视频播放与预览
- 支持主流格式（MP4、MKV、MOV、AVI 等）
- 时间轴缩略图预览，快速定位画面
- 精确 Seek，点击时间轴任意位置即时跳转

#### 3.2 裁剪导出
- 在时间轴上拖拽选取需要保留的片段
- 支持精确到帧的起止时间设置
- 一键导出，自动补全音频时间戳对齐
- 支持硬字幕烧录（将字幕嵌入画面）或软字幕封装

#### 3.3 AI 字幕生成
基于 OpenAI Whisper 的本地语音识别：
- **自动识别**：上传视频 → 自动提取音频 → Whisper 转写 → 生成时间轴字幕
- **多语言支持**：中文、英语、日语、韩语等（取决于 Whisper 模型）
- **VAD 语音分割**：自动过滤静音段，提升识别效率
- **批量处理**：支持同时导入多个视频批量生成字幕

#### 3.4 字幕编辑
- **可视化时间轴**：每条字幕以时间块形式展示，直观可拖拽
- **文本编辑**：双击字幕条目即可修改文字内容
- **时间微调**：拖拽字幕块边缘调整起止时间
- **播放头对齐**：将字幕开始/结束时间一键对齐到当前播放位置
- **字幕拆分**：在当前播放位置将一条字幕拆分为两条
- **整体偏移**：批量将所有字幕提前或延后

#### 3.5 字幕导出
支持三种主流格式导出：
- **SRT** — 通用字幕格式，兼容几乎所有播放器
- **VTT** — Web 视频标准格式
- **ASS** — 高级格式，支持样式定义

#### 3.6 字幕加载
打开视频时自动检测并加载：
1. 视频内嵌软字幕（自动提取）
2. 同目录下同名 `.srt` 文件
3. 之前生成的临时字幕文件

## 界面亮点

- **深色主题**：专为长时间工作设计，减少视觉疲劳
- **左侧图标导航**：68px 纯图标导航栏，模块切换一目了然
- **三栏布局**：视频列表 | 播放器 + 时间轴 | 字幕列表，信息密度高而不乱
- **SVG 图标系统**：全矢量图标，任意缩放清晰锐利
---

## 技术栈

| 组件 | 技术 |
|---|---|
| UI 框架 | Qt 6.8 (Widgets) |
| 视频编解码 | FFmpeg 7.x |
| 录屏采集 | Windows DXGI + WASAPI |
| 语音识别 | OpenAI Whisper (ggml) |
| 繁简转换 | OpenCC |
| 构建系统 | CMake + Ninja |

---

## 系统要求

- **操作系统**：Windows 10 / 11（64 位）
- **运行环境**：无需安装，绿色便携，开箱即用
- ** Whisper 模型**：字幕生成功能需额外下载语音模型文件（首次使用时提示）
